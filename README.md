## Objetivo
Criar uma visão estratégica de indicadores comerciais (KPIs), permitindo a análise rápida do faturamento total, lucratividade e distribuição geográfica das vendas por marcas e continentes.

## Funcionalidades Principais

| Recurso | Descrição |
| :--- | :--- |
| **KPIs de Vendas** | Monitoramento de Faturamento, Lucro, Volume de Vendas e Base de Clientes. |
| **Análise Temporal** | Gráfico de linha para acompanhamento da evolução do faturamento ao longo do tempo. |
| **Análise por Marca** | Comparativo de performance de faturamento entre diferentes marcas do portfólio. |
| **Análise Geográfica** | Mapa interativo com visualização da distribuição de receita por continentes. |

## Tecnologias Utilizadas
* **Power BI:** Ferramenta de visualização utilizada para a criação dos dashboards interativos.
* **Modelagem de Dados:** Estruturação das bases de dados para garantir a correta integração e visualização dos indicadores de vendas sem a necessidade de cálculos complexos.
